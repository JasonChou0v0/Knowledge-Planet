# 知识星球-ASP.NET课程设计

#### 介绍
【ASP.NET课程设计】知识星球是一款面向大学生的知识共享平台，用户可以对自有的学习资料进行上传共享，也可以在平台查找自己需要的学习资料。管理员可以对用户上传的资料进行审核，并管理已通过审核的资料。
